# 📘 Glosarium Artificial Intelligence & Data Science

Repository ini berisi kumpulan istilah penting dalam bidang **Artificial Intelligence (AI)**, **Machine Learning**, **Data Science**, **Microsoft Fabric**, dan **Business Intelligence** yang disusun berdasarkan urutan alfabet.

Glosarium ini dapat digunakan sebagai referensi belajar bagi mahasiswa, dosen, peneliti, maupun praktisi yang ingin memahami istilah-istilah teknis dalam dunia AI dan analisis data.

# A

- **Algoritma**  
  Sekumpulan instruksi atau aturan yang terdefinisi dengan baik untuk memecahkan suatu masalah.

- **Artificial Intelligence (AI)**  
  Bidang ilmu komputer yang bertujuan mengembangkan sistem yang mampu meniru kecerdasan manusia.

- **API**  
  Antarmuka yang memungkinkan dua aplikasi atau sistem saling berkomunikasi.

- **Automatic Code Completion**  
  Fitur editor kode yang memberikan saran penyelesaian kode secara otomatis.

---

# B

- **Batch Scoring**  
  Proses menghasilkan prediksi terhadap sejumlah besar data secara bersamaan.

- **Bias**  
  Kesalahan sistematis pada data atau model yang menyebabkan hasil prediksi tidak akurat.

- **Business Intelligence (BI)**  
  Teknologi dan proses untuk mengumpulkan, menganalisis, dan menyajikan data guna mendukung pengambilan keputusan.

---

# C

- **Classification**  
  Model Machine Learning yang digunakan untuk memprediksi kategori.

- **Clustering**  
  Teknik pengelompokan data berdasarkan kemiripan karakteristik.

- **CSV (Comma-Separated Values)**  
  Format penyimpanan data tabular dengan pemisah koma.

---

# D

- **Data Science**  
  Ilmu yang memanfaatkan statistik, algoritma, dan komputasi untuk memperoleh informasi dari data.

- **Data Engineering**  
  Proses membangun sistem pengumpulan, penyimpanan, dan pengolahan data.

- **Data Visualization**  
  Penyajian data dalam bentuk grafik atau visual agar lebih mudah dipahami.

- **Deployment**  
  Proses menerapkan model Machine Learning ke lingkungan produksi.

---

# E

- **ETL (Extract, Transform, Load)**  
  Proses mengambil, mengubah, dan memuat data ke sistem tujuan.

- **Exploratory Data Analysis (EDA)**  
  Tahapan awal analisis data untuk memahami pola, karakteristik, dan anomali.

---

# F

- **Feature Engineering**  
  Proses membuat fitur baru yang lebih informatif untuk meningkatkan performa model.

- **Forecasting**  
  Teknik prediksi nilai masa depan berdasarkan data historis.

---

# G

- **Generative AI**  
  Cabang AI yang mampu menghasilkan konten baru seperti teks, gambar, audio, maupun kode.

---

# H

- **Histogram**  
  Grafik distribusi frekuensi data numerik.

- **Hyperparameter**  
  Parameter yang ditentukan sebelum proses pelatihan model dimulai.

---

# I

- **Insight**  
  Informasi atau pemahaman yang diperoleh dari hasil analisis data.

- **Imputasi**  
  Teknik mengisi data yang hilang dengan nilai estimasi.

---

# J

- **JSON**  
  Format pertukaran data yang ringan dan mudah dibaca manusia.

---

# K

- **Korelasi**  
  Hubungan statistik antara dua variabel.

---

# L

- **Lakehouse**  
  Arsitektur data modern yang menggabungkan Data Lake dan Data Warehouse.

- **Library**  
  Kumpulan fungsi atau kode yang dapat digunakan kembali.

---

# M

- **Machine Learning**  
  Cabang AI yang memungkinkan komputer belajar dari data tanpa diprogram secara eksplisit.

- **Model**  
  Representasi matematis yang digunakan untuk melakukan prediksi.

- **MLflow**  
  Framework untuk mengelola siklus hidup Machine Learning.

---

# N

- **Normalisasi**  
  Teknik mengubah skala data ke rentang tertentu.

- **Notebook**  
  Lingkungan interaktif untuk menjalankan kode dan dokumentasi secara bersamaan.

---

# O

- **One-Hot Encoding**  
  Teknik mengubah data kategorikal menjadi data numerik.

- **ONNX**  
  Format standar untuk pertukaran model Machine Learning.

---

# P

- **Power BI**  
  Platform visualisasi data dari Microsoft.

- **Principal Component Analysis (PCA)**  
  Teknik reduksi dimensi untuk menyederhanakan data.

- **Predictive Analysis**  
  Analisis yang digunakan untuk memprediksi kejadian di masa depan.

---

# R

- **Regression**  
  Model Machine Learning yang memprediksi nilai numerik.

- **R-Squared (R²)**  
  Metrik untuk mengukur kualitas model regresi.

---

# S

- **Scikit-learn**  
  Library Python populer untuk Machine Learning.

- **Spark Compute**  
  Sumber daya komputasi berbasis Apache Spark.

- **Standardisasi**  
  Teknik mengubah data agar memiliki rata-rata 0 dan standar deviasi 1.

---

# T

- **Training**  
  Proses melatih model Machine Learning menggunakan data.

- **Time-Series Data**  
  Data yang disusun berdasarkan urutan waktu.

---

# U

- **Univariat**  
  Analisis yang hanya melibatkan satu variabel.

---

# V

- **Variabel Target**  
  Variabel yang menjadi tujuan prediksi model.

- **Version Control**  
  Sistem yang mencatat perubahan pada file selama pengembangan.

---

# W

- **Workspace**  
  Ruang kerja untuk mengelola proyek, data, dan aset analitik.

- **Workload**  
  Jumlah pekerjaan komputasi yang harus diproses sistem.
